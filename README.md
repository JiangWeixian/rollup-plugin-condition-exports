# rollup-plugin-condition-exports
*auto set conditions exports on package file*

## install

```console
pnpm add rollup-plugin-condition-exports
```

## usage

> **💡 NOTE**  
make sure `tsconfig.json dist` same as `rollup.config.js output.dir`

## development

- **Setup** - `pnpm i`
- **Build** - `pnpm build`
